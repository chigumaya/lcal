# lcal

## これは何？

cal コマンドに似てるけどちょっと違うカレンダーコマンド。

- カラー表示
- 日本の祝日を表示

## インストール

まず、LuaまたはLuaJITのインストールが必要。macOS (Homebrew) の場合、

```
brew install lua
brew install luajit
```

どちらでも可。Luaの標準実装で高速なのがlua。Luaの仕様としては古め(5.1)だけどめちゃくちゃ高速なのがluajit。それ以外のOSでも似たようなパッケージ名になってるはず。

次に、`lcal.lua` の1行目の
```
#!/usr/local/bin/lua
```
の行をインストールしたインタプリタの場所に書き換える。macOS (Homebrew / Apple Silicon)では `#!/opt/homebrew/bin/lua` または `#!/opt/homebrew/bin/luajit` のように書き換える。多くのLinux環境ではおそらく `#!/usr/bin/lua` または `#!/usr/bin/luajit` になるだろう。

あとは書き換え後のスクリプトをパスの通ったところにコピーして実行権限を付与すればよい。

```
install -m755 lcal.lua /path/to/lcal
```


## 使い方

```
% lcal [オプション] [年月]
```

### オプション
```
-3      指定した月とその前後の3ヶ月を表示
-y      指定した年を12ヶ月表示
-n      祝日名などを表示しない
-t      標準出力がttyでない場合でもエスケープシーケンスを出力
-h      ヘルプ
```

### 表示する年月の指定

cal コマンドは MM YYYY の形式でしか指定できないが、lcal は以下の形式で指定可能。無指定時は今月分のカレンダーを表示する。

```
MM YYYY	    calと同じ
YYYY MM     日本語での一般的な日付表記
MM          次のMM月を表示
            ex. 5月に実行した場合、MMが1-4なら来年のMM月、5-12なら今年のMM月を表示
YYYY        指定した年の12ヶ月分
```

## カスタマイズ

`~/.lcal` というファイルで表示させる内容をカスタマイズできる。このファイルは Lua の文法で記述する。

### 記念日

祝日でない日をカレンダーに表示できる。

```
memorial_day = {
  { name="誕生日", month=1, day=23, first=1980 },
  { name="飲み会", month=5, day=15, year=2026 },
}
```

この場合、1980年以降の毎年1/23を「誕生日」、2026/5/15を「飲み会」として表示する。

`memoriarl_day` に登録した日は祝日と異なるイベントとして扱われる(振替休日の判定対象にならない、画面表示の色が異なるなど)。

画面表示の色を休日と同じにしたい場合は、`memorial_day` ではなく `holiday` または `holiday2` に追加する。かならず `table.insert()` を使って既存のテーブルに追加すること。どちらも休日扱いされるが、`holiday` は日曜日と重なった場合は翌日が振替休日として扱われ、`holiday2` は振替休日にならない。

```
table.insert(holiday2, { name="埼玉県民の日", month=11, day=14, first=1971 })
```

記念日テーブルで指定する各キーは以下のとおり。

- `name`
  - その記念日の名前
- `month`
  - 月
- `day`
  - 日
  - `{m,n}` のように記載した場合はその月のm番目の第n曜日(0=日曜、6=土曜)
- `year`
  - 年 (その年だけ)
- `first`
  - 年 (その年以降毎年)
- `last`
  - 年 (その年まで毎年)

たとえば、以下のように記述した場合は「2012年から2023年までの1月第2月曜」を表す。
```
{ name="foo", month=1, day={2,1}, first=2012, last=2023 }
```

このほか、年を引数にして日付を返す関数を登録することで、イースターのように年によって異なる日に発生するイベントにも対応可能。詳しくはソースを参照。

### その他

`~/.lcal` に以下を指定すると曜日ラベルがアルファベットから漢字になる。

```
weekday_labels = { "日", "月", "火", "水", "木", "金", "土" }
```

## 制限

- 日本以外の国のカレンダー表示には非対応
  - 振替休日とオセロ休日の制度がない国であれば、これらの制度がはじまった日を遠い未来に設定した上で、祝日テーブルを置き換えれば対応できるはずだが未確認
- 日本においてグレゴリオ暦が採用された1873年1月以降のみ対応
- 古い時代(とくに戦前)の祝日は抜けや漏れがある可能性がある
- 現時点では春分/秋分の日は2150年までの対応
- locale に応じて表示を変えたりはしない

- `~/.lcal` には任意のLuaコードを書くことができ、それを無条件に信用して実行する。外部から汚染されたコードが注入される可能性は低いと思われるが注意すること。
